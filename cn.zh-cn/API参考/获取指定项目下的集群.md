# 获取指定项目下的集群<a name="cce_02_0239"></a>

## 功能介绍

该API用于获取指定项目下所有集群的详细信息。

## 调试<a name="atuogenerate_1"></a>

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=CCE&api=ListClusters)中调试该接口，支持自动认证鉴权。API Explorer可以自动生成SDK代码示例，并提供SDK代码示例调试功能。

## URI

GET /api/v3/projects/\{project\_id\}/clusters
