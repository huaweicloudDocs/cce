# 集群唤醒<a name="cce_02_0375"></a>

## 功能介绍

集群唤醒用于唤醒已休眠的集群，唤醒后，将继续收取控制节点资源费用。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=CCE&api=AwakeCluster)中调试该接口。

## URI

POST /api/v3/projects/\{project\_id\}/clusters/\{cluster\_id\}/operation/awake
