# 重置节点<a name="ResetNode"></a>

## 功能介绍

该API用于在指定集群下重置节点。

>![](public_sys-resources/icon-note.gif) **说明：** 
>集群管理的URL格式为：https://Endpoint/uri。其中uri为资源路径，也即API访问的路径。

## 调试<a name="atuogenerate_1"></a>

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=CCE&api=ResetNode)中调试该接口，支持自动认证鉴权。API Explorer可以自动生成SDK代码示例，并提供SDK代码示例调试功能。

## URI

POST /api/v3/projects/\{project\_id\}/clusters/\{cluster\_id\}/nodes/reset
