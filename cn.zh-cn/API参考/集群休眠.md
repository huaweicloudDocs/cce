# 集群休眠<a name="cce_02_0374"></a>

## 功能介绍

集群休眠用于将运行中的集群置于休眠状态，休眠后，将不再收取控制节点资源费用。

## 接口约束

1、集群休眠后，将无法在此集群上创建和管理工作负载等资源。

2、按需付费集群休眠后，将暂停收取控制节点资源费用，集群所属的节点、绑定的弹性IP、带宽等资源按各自的计费方式（“包年/包月”或“按需付费”）进行收费。
