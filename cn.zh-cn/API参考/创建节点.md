# 创建节点<a name="cce_02_0242"></a>

## 功能介绍

该API用于在指定集群下创建节点。

>![](public_sys-resources/icon-note.gif) **说明：** 
>-   若无集群，请先[创建集群](创建集群.md)。
>-   集群管理的URL格式为：https://Endpoint/uri。其中uri为资源路径，也即API访问的路径。

## 接口约束

仅支持创建KVM虚拟化类型的节点，非KVM虚拟化类型的节点创建后无法正常使用。

## 调试<a name="atuogenerate_1"></a>

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=CCE&api=CreateNode)中调试该接口，支持自动认证鉴权。API Explorer可以自动生成SDK代码示例，并提供SDK代码示例调试功能。

## URI

POST /api/v3/projects/\{project\_id\}/clusters/\{cluster\_id\}/nodes
